# Teacher workspace family

References: `design-references/weelmat-generator-workspace-concept.png` and `design-references/weelmat-account-workspace-concept.png`.

- Use one workspace shell across account, history, submission, premium hub, and generators.
- Generator: four-step orientation, open form groups, compact day rail/editor, sticky progress summary.
- Account/history: slim navigation rail desktop, top tabs mobile, week strip, filterable file list/table, status rail.
- Submission: focused Details → File → Confirm composition. Do not duplicate full submission forms across pages.
