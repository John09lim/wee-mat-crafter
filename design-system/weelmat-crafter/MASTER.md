# WeeLMat Crafter Design System

The canonical product system is `../weelmat/MASTER.md`. All WeeLMat Crafter routes must follow it. This compatibility file exists so older page-family lookups do not fall back to the retired slate/blue theme.

## Locked identity

- Forest `#173F2A`
- Action green `#236130`
- Gold `#D6A73D`
- Cream `#F6F0E7`
- Paper `#FFFCF7`
- Ink `#142019`
- Warm border `#D8D0C4`
- Display `Lora`; UI/body `Raleway`

Use open editorial layouts for marketing and table/list-led workspaces for product screens. Minimum 44px controls, visible focus, inline form errors, reduced motion, mobile records for wide tables, and no duplicate workspace footers.
